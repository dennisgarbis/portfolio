Greek Tavern

How It Was Made

I'm Greek. I like Greek food. So what else would I do but make a Greek Restaurant and name it after a Greek God from whom my name originates.

Project Bootstrap

This project was to create a responsive website. I decided to use Bootstrap as my CSS framework I got ideas from getbootstrap, along with other sites such as W3Schools and created a Greek restaurant. This was my first time using a framework, and I found it to be not as challenging as I though it would be. Using bootstraps classes turned out to be an easy way to make the site responsive. There are many components to choose from, and these were the ones I ended up using on my site:

  • navbar
  • carousel
  • contact form
  • list group

How It Started

I made a logo for my Photoshop class for a Greek island touring company. It was called Cyclade's Tour. I took that logo and replaced the name with what you see here.

The wireframe was a really rough sketch of what I though the menu would look like. The header and footer came out to looking something like what I had originally drawn, but the main content did not. I did use three columns, but they were separated with rows. I also never included an image toward the bottom that I had intended on. The rest of the site I did off the top of my head.

The rest of the site I did off the top of my head. I had ideas as to what will go where, but I changed the layout often while I was coding it out. I also created a small movie using After Effects. I had a real fun time with After Effects and wanted to continue using it in more in my projects.

I included a carousel and testimonials on the homepage and a contact form on the contact page. This was my first time using Bootstrap and I found it very easy for everything to be responsive.

The About Us page had a gallery that is fully respoinsive, again thanks to using Bootstrap. Have I mentioned how easy it was to set everything up with Bootstrap?

The most difficult part of the entire website was creating the image below. It is found on the About Us page and for some reason it was giving me difficulties. After a bit of erasing code and trying again, I got it to work.

The Finished Product

The project took me about a month to finish. I spent most of my time adjusting the table showing the hours of operation, as well as the carousel with the food. The biggest sticking point there was how to have the title of the food show on the screen. I learned a lot using Bootstrap. Once I got the hang of how the code is written out, it was pretty straightforward. Though I enjoy using flexbox, and understand it pretty well, Bootstrap will be something I will be using in the future again.
