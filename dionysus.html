Project Bootstrap

This project was to create a responsive website. I chose to use Bootstrap has my CSS framework to create a Greek restaurant. This was my first time using a framework, and I found to be not as challenging as I though it would be. Using bootstraps classes turned out to be an easy way to make the site responsive. There are many components to choose from, and these were the ones I ended up using on my site:

  • navbar
  • carousel
  • contact form
  • list group
