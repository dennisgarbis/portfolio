Website Redesign

There are plenty of outdated websites out there. Many have usability issues along with other issues. This project asked us to search out a website that needed a complete makeover. We had to find usability issues, create a customer persona and journey map, and then redesign three pages. It was a great learning experience to design a website with the customer in mind and to see how it would benefit them most.

Project Scope

The task for this project was to sketch out a redesigned site, build a wireframe, and then create interactive prototypes. The goal was to design our websites to ensure optimal website usability, combining user goals with current design trends.

The first step had us completing moderated remote usability testing, and taking notes on our findings. We then created a persona and journey map based on our test findings. Sketches were then created based of our created persona and journey map. Using Balsamiq, I created wireframes based of my hand-drawn sketches, which I then followed by using Axure RP to create interactive prototypes.
